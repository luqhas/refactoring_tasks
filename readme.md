### Требования:
- Python 3.8+
- pip (менеджер пакетов Python).
- Любой редактор кода.

---

### Перед запуском рекомендуется:
- Создать виртуальную среду, чтобы изолировать связи между проектами на устройстве.
- Команда для создания вирутальной среды: ```py -m venv 'название папки'```.
- Активровать среду, если она не активировалось по умолчанию рабочей средой.

---

### Установка зависимостей.

- Перейти в главную директорию, и оттуда запустить команду ```py pip install -r requirements.txt```

### Как запустить

- В каждрй папке имеется файл с соответствующим названием.
